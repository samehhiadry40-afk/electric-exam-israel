# מאגר שאלות בחינת רישוי חשמל — גרסה 7.1

אתר הכנה לבחינת רישוי חשמלאי מהנדס.

## תכונות
- 👤 **חשבון בענן (Google Sign-In)** — התחבר עם חשבון Google שלך וההיסטוריה והסטטיסטיקות מסונכרנות אוטומטית בין מחשב ונייד
- 📚 **מצב לימוד** — 958 שאלות, ממוינות לפי נושאים, עם הסברים ורמזים
- 📝 **סימולציית מבחן** — 50 שאלות אקראיות, שעון 3 שעות, ציון עובר 80
- 📊 **סטטיסטיקה פר-שאלה** — מעקב אחרי כמה פעמים ענית על כל שאלה ואחוז ההצלחה
- 🚨 **שאלות בעייתיות + טיפים** — האתר מזהה שאלות שאתה מתקשה איתן ונותן טיפים ללימוד מחדש
- 📖 **חומר תיאורטי** — סיכומים מקיפים לכל הנושאים
- 📡 **עבודה במצב לא-מקוון** — שינויים שעשית ללא רשת ייסנכרנו אוטומטית כשהרשת תחזור (Firestore offline cache)

## פרטיות וסנכרון
הנתונים שלך (היסטוריה, סטטיסטיקות) נשמרים בחשבון Firebase האישי שלך, מסונכרנים אוטומטית בין כל המכשירים בהם נכנסת עם אותו חשבון Google. חוקי הגישה ב-Firestore (`firestore.rules`) מבטיחים שרק אתה רואה את הנתונים שלך — אף משתמש אחר אינו יכול לגשת אליהם. ניתן לאפס/למחוק את הנתונים בכל עת מתוך עמוד "פרופיל".

בכניסה הראשונה עם Google, אם קיימים פרופילים מקומיים מהגרסה הישנה (localStorage), תוצג לך אפשרות לייבא אותם פעם אחת לענן.

## הגדרת Firebase (חד-פעמית)
לפני שהאתר יוכל להתחבר עם Google, צריך לחבר אותו לפרויקט Firebase משלך:

1. היכנס ל-[Firebase Console](https://console.firebase.google.com) וצור פרויקט חדש (חינם, ללא כרטיס אשראי)
2. ב-**Authentication → Sign-in method**: הפעל את ספק **Google**
3. ב-**Cloud Firestore**: צור מסד נתונים, בחר Production mode
4. הוסף Web App לפרויקט והעתק את אובייקט `firebaseConfig`. הדבק אותו ב-`index.html` במקום הערכים הזמניים שמתחילים ב-`PASTE_...`
5. ב-**Firestore → Rules**: העתק את התוכן של `firestore.rules` (בתיקייה הזו) והדבק שם
6. ב-**Authentication → Settings → Authorized domains**: הוסף את הדומיין שמארח את האתר (למשל `samehhiadry40-afk.github.io`)

## דרישת אירוח
האתר חייב להתארח ב-HTTPS (לא מ-`file://`). אופציות חינמיות:
- **GitHub Pages** — הכי פשוט, האתר כבר מחובר לרפו
- **Netlify / Vercel / Cloudflare Pages** — drag-and-drop של התיקייה
- **שרת מקומי לבדיקה**: `python -m http.server 8000` ואז `http://localhost:8000`
