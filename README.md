# מאגר שאלות בחינת רישוי חשמל — גרסה 7.1

אתר הכנה לבחינת רישוי חשמלאי מהנדס.

## תכונות
- 👤 **פרופילים אישיים** — צור פרופיל ושמור את ההיסטוריה והסטטיסטיקה שלך (נשמר מקומית בדפדפן)
- 📚 **מצב לימוד** — 958 שאלות, ממוינות לפי נושאים, עם הסברים ורמזים
- 📝 **סימולציית מבחן** — 50 שאלות אקריות, שעון 3 שעות, ציון עובר 80
- 📊 **סטטיסטיקה פר-שאלה** — מעקב אחרי כמה פעמים ענית על כל שאלה ואחוז ההצלחה
- 🚨 **שאלות בעייתיות + טיפים** — האתר מזהה שאלות שאתה מתקשה איתן ונותן טיפים ללימוד מחדש
- 📖 **חומר תיאורטי** — סיכומים מקיפים לכל הנושאים

## פרטיות
כל נתוני הפרופיל (היסטוריה, סטטיסטיקות) נשמרים אך ורק ב-`localStorage` של הדפדפן המקומי שלך — אין שרת ואין שיתוף בין מכשירים. ניתן לאפס/למחוק את הנתונים בכל עת מתוך עמוד "פרופיל".
